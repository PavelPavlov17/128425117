# Оптимизация на верига за доставки (Supply–Demand Chain Optimization)

## Описание
Този проект представлява проста конзолна програма на езика C за оптимизация на веригата за доставки в производствено предприятие.

Целта на програмата е да минимизира общата цена за снабдяване, като избира доставчици с най-ниска цена на единица суровина, докато се покрие зададеното търсене на фабриката.

## Функционалност
- Въвеждане на брой доставчици
- Въвеждане на цена и налично количество за всеки доставчик
- Въвеждане на общото търсене на фабриката
- Сортиране на доставчиците по цена (възходящ ред)
- Оптимално разпределение на количествата
- Финална справка за използваните доставчици и общата цена
- Предупреждение при недостатъчно общо предлагане

## Използван алгоритъм
Използван е greedy алгоритъм:
1. Доставчиците се сортират по цена
2. Закупува се максимално възможното количество от най-евтиния доставчик
3. Процесът продължава, докато търсенето бъде покрито или доставките се изчерпат

Алгоритъмът гарантира минимална крайна цена при дадените ограничения.
